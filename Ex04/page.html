Aqui está um texto breve do renomado autor de literatura Howard Phillips Lovecraft

"Eu não tenho medo do desconhecido. Eu tenho medo do conhecido. O desconhecido é um mistério, o conhecido é uma ameaça. O desconhecido é um desafio, o conhecido é uma prisão. O desconhecido é uma aventura, o conhecido é uma rotina. O desconhecido é uma oportunidade, o conhecido é uma limitação. O desconhecido é um sonho, o conhecido é um pesadelo."
